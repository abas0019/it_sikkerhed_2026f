## Grænseværditest (Boundary Value Testing)

Grænseværditest handler om at teste værdier lige ved grænsen, fordi fejl ofte opstår dér.

Hverdagseksempel (elbil / batteri):
Hvis reglen er:
- Batteri > 0% = bilen kan køre
- Batteri = 0% = bilen kan ikke køre
